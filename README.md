# bookbot

BookBot is my first [Boot.dev](https://www.boot.dev) project!

It is a simple program that takes in a path to a .txt file as input and outputs the number of words in the .txt file and which letters in the alphabet were inside the .txt file however many times.


This is my first ever actual project on python and the first repisotory i am publishing on github.com. As it can be interpreted from the text above, all of my knowledge to produce this program came from boot.dev, so thanks to them for helping me in my programming journey!
